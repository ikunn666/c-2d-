# NARUTO - 本地双人格斗游戏

基于 C++ 和 raylib 图形库开发的本地双人 2D 格斗游戏。项目采用面向对象设计，实现了状态机控制、碰撞检测以及火影忍者主题的角色技能。

项目开发全流程采用 Git 进行版本控制，保持了良好的提交历史与代码规范。

---

## 游戏操作说明

游戏采用单键盘双人对战模式，操作布局如下：

### 玩家 1（P1 - 左侧）

- 移动：A（向左）/ D（向右）
- 跳跃：K
- 普通攻击：J
- 技能一：U
- 技能二：I
- 替身术：L

### 玩家 2（P2 - 右侧小键盘）

- 移动：←（左箭头）/ →（右箭头）
- 跳跃：2（小键盘）
- 普通攻击：1（小键盘）
- 技能一：4（小键盘）
- 技能二：5（小键盘）
- 替身术：3（小键盘）

### 全局控制

- 胜负判定：任意一方生命值归零或时间60s倒计时结束时游戏立刻结束。
- 重新开始：游戏结束后，按下键盘上的 R 键可直接重启对局。

## 编译与运行指南

本项目内置了完整的 raylib 依赖库，并提供了两种编译构建方式：

### 选项 A：VS Code 一键调试

项目已配置好 `.vscode` 文件夹下的环境：

1. 使用 VS Code 打开 NARUTO 根目录。
2. 确保已安装 C/C++ 扩展，且本地配置了 MinGW 环境。
3. 按下键盘上的 F5 键，即可自动触发 tasks.json 进行编译并启动调试。

### 选项 B：使用批处理脚本

工程根目录下内置了一键编译脚本：

1. 双击运行根目录下的 `build-MinGW-W64.bat`。
2. 脚本会自动调用 MinGW 编译器构建项目，并在编译完成后运行游戏。
